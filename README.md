# Projeto de Regressão Linear - Notas vs Horas de Estudo

## 📌 Sobre o projeto
Este projeto analisa a relação entre **horas estudadas** e **notas de exame** utilizando regressão linear. Foi desenvolvido como parte do meu portfólio em Análise de Dados.

## 📊 Principais conclusões
- Correlação de **0,994** entre horas e notas (relação quase perfeita)
- Cada hora extra de estudo aumenta a nota em **~10,4 pontos percentuais**
- O modelo explica **98,7%** da variação das notas (R² = 0,987)

## 🛠️ Tecnologias utilizadas
- Python
- Pandas (manipulação de dados)
- Matplotlib (visualização)
- Google Colab

## 📁 Como executar
1. Abra o notebook no Google Colab
2. Execute as células em ordem
